## Agent-Specific Considerations (Goal Memory / Metacognition)

For metacognitive assessment work, pay special attention to:
- **Assessment scope** - what aspects of workflow state to evaluate
- **Goal alignment** - how to measure progress toward objectives
- **Unknown tracking** - what knowledge gaps remain
- **Workflow health** - what indicators suggest problems
- **Transition readiness** - what determines phase advancement readiness

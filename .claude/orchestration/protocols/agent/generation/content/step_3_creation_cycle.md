# Generation Strategy Development

## Tasks

1. **IDENTIFY** artifact type and applicable standards
2. **SELECT** appropriate templates and patterns
3. **PLAN** incremental build approach with checkpoints
4. **DEFINE** validation criteria for each increment

| Plans | STRUCTURE-DETAIL-VALIDATE | Phases → Tasks → Check |
| Content | CONCEPT-DEVELOP-REFINE | Theme → Create → Enhance |

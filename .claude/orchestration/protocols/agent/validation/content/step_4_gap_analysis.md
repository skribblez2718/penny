# Gap Analysis and Remediation Guidance

## Gap Types

- Missing elements or incomplete sections
- Non-conforming aspects requiring correction
- Areas where quality is below threshold
- Edge cases not adequately addressed

## Gap Documentation

For each gap:
```
GAP: {description}
SEVERITY: {CRITICAL|HIGH|MEDIUM|LOW}
LOCATION: {where in artifact}
IMPACT: {what's affected}
REMEDIATION: {how to fix}
EFFORT: {estimate}
```

## Prioritization

| Priority | Criteria |
|----------|----------|
| P0 | Blocks deployment/approval |
| P1 | Significant quality impact |
| P2 | Moderate quality impact |
| P3 | Minor improvement |

## Remediation Guidance

Provide actionable guidance:
- Specific corrective actions
- Reference applicable standards
- Estimate remediation effort
- Suggest preventive measures

## Completion Criteria

- [ ] Gaps identified
- [ ] Severity assigned
- [ ] Prioritization complete
- [ ] Remediation guidance provided
- [ ] Ready for gate decision

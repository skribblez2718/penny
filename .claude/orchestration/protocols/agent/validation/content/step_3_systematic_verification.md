# Systematic Verification

## Execute Checks by Artifact Type

### Technical Artifacts
- Functionality: Does it work correctly?
- Security: Are security practices followed?
- Performance: Does it meet performance targets?
- Maintainability: Is code clean and documented?
- Test Coverage: Are tests adequate?

### Creative Artifacts
- Alignment: Does it match the brief?
- Audience: Is it appropriate for audience?
- Tone: Is voice/style consistent?
- Structure: Is it well-organized?

### Decisions
- Criteria Satisfaction: Does it meet requirements?
- Risk Assessment: Are risks acceptable?
- Goal Alignment: Does it serve objectives?

### Plans
- Completeness: Is everything covered?
- Feasibility: Can it be executed?
- Timeline: Is it realistic?

## Document Findings

For each check:
- Record specific instances of non-compliance
- Capture test results and measurements
- Note failures AND marginal passes
- Identify patterns in issues

## Score Against Criteria

- Assign objective scores where applicable
- Weight criteria by importance
- Calculate aggregate quality metrics
- Determine confidence level

## Completion Criteria

- [ ] All criteria checked
- [ ] Findings documented
- [ ] Scores assigned
- [ ] Ready for gap analysis

# Lit Feature Index

- [Lit Documentation Lookup](lit.md): Live-lookup entry point — canonical base (https://lit.dev/docs/), section→path map, and version check; plus styling (Tailwind) and secure-coding cross-refs. Fetch current pages at runtime rather than trusting a static list (see `docs/agents/coding/library-docs.md`).
- **Styling:** Tailwind CSS (`docs/agents/coding/tailwind/AGENTS.md`) is the default CSS framework — see the Lit + Tailwind shadow-DOM integration in `.pi/skills/code/resources/web-ui.md`.
- **Security:** consult `docs/agents/coding/security/AGENTS.md`; Lit-specific XSS/secrets/API notes are in the "Secure Coding" section of [lit.md](lit.md#secure-coding-project-standards).

> **Lit is the documented default UI approach for this project.** Build web
> application UIs as custom Lit web components. See
> `docs/agents/coding/conventions.md` for the default-selection policy and
> `.pi/skills/code/resources/web-ui.md` for patterns and gotchas.

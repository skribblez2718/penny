derivation skill — execution flow

Validated against orchestration/playbooks/derivation.py (DerivationMachine) · runs on the shared orchestration engine. A verify-only, source-agnostic independence gate: two phases, auto-routed by the SHAPE of constraints.sources (no extra flag). It judges, never fixes.
Corpus-completeness guard (no vacuous pass): independence is judged against the corpus — a manifest that omits the restricted source a lesson was rebuilt from would "pass" with nothing restricted to compare against. A clean-room rebuild's source MUST be present, tagged role: "coverage-reference"; if absent, the reviewer returns UNCERTAIN (escalates to a human), never a clean verdict. Pass the gate corpus here, not a shipped manifest that deliberately omits restricted artifacts.

Phase 1 — gathering (echo dynamic fan; runs ONLY when sources is a directory)

  • LOCAL, READ-ONLY inventory: one branch per scannable source file (.md/.txt/.rst/.text, mirroring prefilter.py; large files sharded). Each branch reports a grounded license/bucket call (identifier + evidence snippet + confidence) and a headings-only outline. NEVER fetches, NEVER discovers new sources, NEVER mutates the corpus, NEVER sets the verdict.
  • Bounded: constraints.max_fan_width (default 8) per round; multi-round batching bounded by ctx.max_iterations. NOT escalatable — a shortfall (zero scannable files, or budget spent before 100% coverage) is a terminal error, never a partial-corpus pass-through.
  • Output: exactly one prefilter.py-compatible manifest.json in a run-scoped 0o700 workdir ({tempdir}/derivation-{session_id}/ or constraints.gather_workdir) — never inside the caller's sources — plus one "<sid> Gather Provenance" mempalace drawer (best-effort, non-fatal).
  • Fail-safe licensing: a non-unknown license with no evidence snippet is downgraded to unknown (unknown ⇒ restricted); a bucket with no marker defaults to "" — never fabricated from the license.

Phase 2 — reviewing (annie; two tiers, UNCHANGED by gather)

  • Tier-1 (deterministic): scripts/prefilter.py — per-source verbatim/n-gram overlap. A hard breach may short-circuit toward DERIVATIVE_RISK; a clean report does NOT imply independence.
  • Tier-2 (judgement): resources/rubric.md — Abstraction → Filtration (facts, math, standard notation, merger, scènes à faire filtered out BEFORE comparing) → Comparison across D1–D7, incl. D3's source-identity tell (adopted course/lesson titles / "Lesson N" numbering) and D7 single-source dependence (the decisive originality signal).
  • Evidence-gated: the engine rejects a verdict without the Tier-1 artifact + per-dimension scoring; a flagged dimension must name both a fix and its matched source(s). When gather ran, annie receives the manifest PATH + outline/pairings as pointers only — they never set or bias the verdict.
  • Verdict → caller action: INDEPENDENT ship · NEEDS_REVISION apply fixes, re-run · DERIVATIVE_RISK do not ship, re-author. Reviewer ≠ author: pin the review to a different model where possible.

Other true-to-code notes

  • Only reviewing escalates (ESCALATABLE_STATES = {reviewing}): UNCERTAIN — including a missing coverage-reference source — pauses in awaiting_clarification and resumes at reviewing on clarify.
  • Abort: every non-final state can abort → error; those arrows are omitted for legibility (only gather's terminal shortfall is drawn).

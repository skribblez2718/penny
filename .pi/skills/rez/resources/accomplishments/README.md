# Accomplishments (source of truth — read-only)

Place accomplishment notes here (markdown preferred). Anything usable as
resume evidence belongs here: project write-ups, metrics, wins, brag-doc
entries, review excerpts.

- The rez skill reads every file in this directory **except this README** as
  supplemental evidence for tailoring.
- The skill never modifies these files.
- If this directory is empty, rez proceeds using only the base resume (not an
  error).
